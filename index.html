Commands to learn

  • git init [initializes git]
  • git add [adds a file]
  • git add --all [adds all files]
  • git add *.txt [adds all txt files]
  • git add docs/ [adds all files in docs direcotry]
  • git add docs/*.txt [adds all files in docs direcotry]
  • git commit -a -m "message" [adds and commits with "message"]
  • git reset --soft HEAD^ [resets last commit]
  • git commit --amend -m "message" [add file to last commit]
  • git reset --hard HEAD^ [undoes last commit and all changes]
  • git reset --hard HEAD^^ [undoes last two commits and all changes]
  • git checkout -- filename [discards changes to staging of files]
  • git push -u origin master [origin is repository name, master is local branch]
  • git push [will push to latest -u specification]
  • git pull [pulls changes down]
  • git remote add name adress [adds new remote repos]
  • git remote rm name [removes remote repos]
  • git clone url [will create a local repo]
  • git clone url depository [will create a local repo in a depo of your choice]
  • git remote -v [adds origin remote pointing it to the clone URL]
  • git branch nameofbranch [will create a branch]
